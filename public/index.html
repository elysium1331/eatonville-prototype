Using the Files

Colors

Forest Green, Gold, Cream — the only brand colors.

#1B4332 #C9952A #FAF7F0

Clear space

Keep open space around any lockup at least equal to the height of the seal (already built into these PNGs).

Minimum size

Icons stay legible to 16 px. Below ~120 px, use an icon instead of the full seal — its fine detail will not hold.

·

Website Favicon

Drop the icon set into the site's root and add these lines to the page <head> (Twenty-Seven shown; swap the folder for the Sun if preferred):

<link rel="icon" type="image/png" sizes="32x32" href="icons/twenty-seven/favicon-32.png">
<link rel="icon" type="image/png" sizes="16x16" href="icons/twenty-seven/favicon-16.png">
<link rel="apple-touch-icon" sizes="180x180" href="icons/twenty-seven/app-icon-180.png">
About this library. The seal is the official mark of record and is never altered. SVG files are infinitely scalable and preferred for print and signage; PNG files are for fixed digital sizes. All files use only Forest Green #1B4332, Gold #C9952A, and Cream #FAF7F0. The seal raster shown is the town's current seal. For the full rationale and standards, see the Brand Standards document.
